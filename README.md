# ContentFramework

> A frame what is working as a content slider. It can be used to show information on a webpage, or create learning materials for a LMS system.

## Supported standards

- SCORM 1.2
- SCORM 2004
- WCAG 2.0

## Example

To create content, just modify the content of the slides.md file.

It has to follow the MarkDown language standards. In runtime, it will be rendered as HTML, and splitted to slides.
